Evolution Calendrier

Roadmap pour l'integration des calendriers

✅ Phase 1 & 2 : Import/Export iCal (IMPLEMENTE)

  • Import des calendriers iCal depuis Booking.com et Airbnb
  • Export du calendrier unifie au format iCal
  • Synchronisation automatique et manuelle
  • Blocage manuel de dates
  • Detection automatique de la plateforme source

⏳ Phase 3 : Integration API complete (A VENIR)

Cette phase necessite des partenariats officiels avec les plateformes de reservation.

🏠 Airbnb API
  • Programme requis: Airbnb Connected Apps Partner Program
  • Inscription: https://www.airbnb.com/partner
  • Pre-requis: Societe etablie, produit SaaS fonctionnel, CGU
  • Delai d'approbation: 2-6 mois
🅑 Booking.com Connectivity Partner API
  • Programme requis: Booking.com Connectivity Partner Program
  • Inscription: https://partner.booking.com/connectivity
  • Pre-requis: Channel Manager ou PMS certifie, 50-100 proprietes
  • Delai d'approbation: 3-12 mois
🏡 VRBO / Expedia Partner API
  • Programme requis: Expedia Group Connectivity Partner Program
  • Inscription: https://developers.expediagroup.com/
  • Delai d'approbation: 2-4 mois
⚠️ Alternative recommandee
Pour une integration plus rapide, envisager l'integration avec un Channel Manager existant: Smoobu, Lodgify, Hostaway, ou Guesty. Delai: 1-2 semaines.
